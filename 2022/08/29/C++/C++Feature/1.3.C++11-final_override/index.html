C++ 11：final 和 override 说明符

发表于2022-08-29|更新于2022-08-30|C++
|阅读量:

final

指定某个虚函数不能在派生类中被覆盖，或者某个类不能被派生。
在继承关系中“中途”终止派生类的重载

override

指定一个虚函数覆盖另一个虚函数。
避免重载函数名拼写错误，函数原型不匹配，重写了非虚函数
